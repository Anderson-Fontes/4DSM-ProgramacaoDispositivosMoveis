## 📖 Sobre o Projeto

O **App Scholar** é um sistema mobile voltado para instituições de ensino superior tecnológico. Ele permite a autenticação de usuários, cadastro de informações acadêmicas e consulta de boletins. 

Este repositório contém a **Parte 1** do projeto, focada na construção da Interface de Usuário (UI/UX), navegação e gerenciamento de estados no frontend, utilizando dados simulados (mockados).

**Contexto Acadêmico:**
* **Instituição:** Fatec Jacareí
* **Curso:** Desenvolvimento de Software Multiplataforma (4º Semestre)
* **Disciplina:** Programação para Dispositivos Móveis I
* **Professor:** André Olímpio

## ✨ Funcionalidades (UI/UX)

* **🔐 Autenticação:** Tela de login com validação de campos vazios.
* **📊 Dashboard:** Menu principal com navegação intuitiva em formato de grid colorido.
* **👨‍🎓 Gestão de Alunos:** Formulário completo para cadastro de discentes (Nome, Matrícula, Curso, Email, Endereço completo, etc).
* **👨‍🏫 Gestão de Professores:** Formulário para registro do corpo docente (Titulação, Área de atuação, etc).
* **📚 Gestão de Disciplinas:** Cadastro de matérias com carga horária e professor responsável.
* **📝 Boletim Acadêmico:** Listagem de notas com feedback visual dinâmico (cores indicando aprovação, exame ou reprovação).

## 🛠️ Tecnologias Utilizadas

* **[React Native](https://reactnative.dev/):** Framework principal para desenvolvimento mobile.
* **[Expo](https://expo.dev/):** Plataforma para facilitar o build e testes do app.
* **[React Navigation](https://reactnavigation.org/):** Gerenciamento de rotas e navegação em pilha (Stack Navigation).
* **Hooks Customizados:** Uso intensivo de `useState` para controle de formulários e `useEffect` para carregamento de dados mockados.
* **Vector Icons:** Utilização da biblioteca de ícones do Expo para uma interface mais rica e intuitiva.
